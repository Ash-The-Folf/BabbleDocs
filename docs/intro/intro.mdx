## What is Project Babble?

Project Babble is a <i>lower-face expression tracking</i> system, designed for use with VR headsets. It works by using image input from a camera and outputs corresponding facial expressions, which in turn can be used by external apps, such as VRCFaceTracking and this Resonite mod.
